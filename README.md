# Синтез парных изображений для верификации "селфи-документ"

## Описание проекта

Данный репозиторий содержит реализацию метода синтеза парных изображений для задачи верификации личности по фотографии в документе и селфи. Метод позволяет на основе одного изображения-шаблона селфи человека с документом создавать синтетические пары изображений "селфи-документ" с контролируемыми изменениями возраста и атрибутов лица.

## Основные возможности

- **Аугментация лиц**: генерация возрастных изменений и добавление очков с сохранением идентичности
- **Сегментация лиц**: автоматическое создание масок лиц с использованием MTCNN и SAM
- **Слияние изображений**: перенос аугментированных лиц на шаблоны с помощью диффузионной модели InsertAnything
- **Анализ качества**: оценка синтезированных изображений по метрикам качества и сохранения идентичности

## Структура репозитория

- **models_for_methods:**  Данная директория содержит реализацию компонент, необходимых для предложенного в исследовании метода.
  - `gender_detector.py` - Определение пола

  - `face_detector.py` - Детекция лиц (MTCNN)

  - `face_generator.py` - Генерация аугментированных лиц

  - `face_masker.py` - Создание масок (SAM2)

  - `face_swapper.py` - Замена лиц 

- **config:** Данная директория содержит настройки для соответсвующих компонент пайплайна.
  - `face_detector_config.py` - Настройки детектора лиц

  - `face_masker_config.py` - Настройки маскера

  - `face_swapper_config.py` - Настройки модели замены лиц
  
  - `generator_config.py` - Настройки генератора лиц

- **Метод.ipynb:**  Данный ноутбук содержит демонстрацию этапов синтеза аугментированных изображений с помощью InterFaceGAN и детекцию области лица с использованием MTCNN. Далее идет этап создания масок с помощью SAM и слияние полученных изображений на шаблоне. Также для каждого этапа представлены примеры полученных изображений.

- **Анализ изображений.ipynb:** В данном файле реализованы все метрики, используемые для оценки качества полученных изображений и приведены примеры их подсчета.


## Описание компонентов

### 1. Генерация аугментированных лиц (`face_generator.py`)

Класс `FacesGenerator` использует InterFaceGAN для синтеза изображений лиц с контролируемыми атрибутами:
- `generated_faces` - исходные сгенерированные лица
- `generated_faces_older` - лица с признаками старения
- `generated_faces_glasses` - лица с очками

### 2. Детекция пола (`gender_detector.py`)

Класс `GenderDetector` на основе трансформерной модели определяет пол человека на изображении для фильтрации генерируемых лиц.

### 3. Детекция лиц (`face_detector.py`)

Класс `FaceDetector` использует MTCNN для обнаружения лиц и создания bounding box'ов с возможностью расширения границ.

### 4. Создание масок (`face_masker.py`)

Класс `FaceMasker` на основе SAM (Segment Anything Model) создает точные маски лиц для последующей замены.

### 5. Замена лиц (`face_swapper.py`)

Класс `FaceSwapper` предоставляет функционал для переноса аугментированных лиц на шаблонные изображения.

### 6. Конфигурационные файлы (`config/`)

Содержат настройки параметров для каждого компонента:
- Размеры bounding box'ов

- Пути к моделям
- Параметры API
- Настройки визуализации
